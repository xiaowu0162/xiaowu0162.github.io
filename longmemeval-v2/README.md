# LongMemEval-V2 Website

Static prototype for the LongMemEval-V2 project website.

Target deployment:

`https://xiaowu0162.github.io/longmemeval-v2/`

## Local Preview

```bash
python3 -m http.server 8022
```

Then open `http://localhost:8022/`.

## Notes

- Paper, code, and data links are intentionally marked TODO.
- The data viewer uses a compact sample bundle derived from the Kaggle archive.
- The full Kaggle dataset archive is not included in this website repository.
